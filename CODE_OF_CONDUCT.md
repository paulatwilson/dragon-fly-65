# Code of Conduct

DragonFly 65 uses the Contributor Covenant Code of Conduct, version 2.1.

In short: be respectful, constructive, and welcoming. Harassment, abuse, and discriminatory behavior are not acceptable in this project.

Project maintainers may remove comments, commits, code, issues, or other contributions that do not align with this standard.

For serious conduct concerns, contact the project maintainer privately.
